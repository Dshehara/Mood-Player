#  Mood-Based Music Player

A modern and responsive music player web application that changes UI and playlists based on user mood.


##  Live Demo
 https://dshehara.github.io/Mood-Player/


##  Features

-  Mood-based playlists (Happy, Sad, Chill, Energy)
-  Play / Pause functionality
-  Next / Previous controls
-  Shuffle &  Repeat modes
-  Song duration & progress bar
-  Volume control
-  Scrollable playlist
-  Dynamic UI based on mood
-  Fully responsive (mobile, tablet, desktop)
-  Smooth animations & loading screen

##  Technologies Used

- HTML5
- CSS3 (Flexbox, Animations, Responsive Design)
- JavaScript 


##  Project Structure
mood-player/

##  How to Run Locally

1. Clone the repository
2. Open index.html in your browser
